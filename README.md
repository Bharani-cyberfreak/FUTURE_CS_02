# Phishing Email Detection & Awareness System

## Task 2 – Future Interns (Cyber Security)
### Track Code: CS

## Objective
To detect phishing emails using key indicators such as spoofed senders, fake domains, and malicious links. This project also creates user awareness to prevent email-based cyber attacks.

## Tools Used
- Python
- Pandas
- Public phishing email samples
- Email header analysis

## Key Features
- Identify phishing indicators
- Classify emails as Safe / Suspicious / Phishing
- Explain phishing techniques in simple language
- Provide prevention and awareness guidelines

## How to Run
1. Install Python (latest version)
2. Install dependencies:
python -m pip instal pandas
3. Navigate to scripts folder:
cd scripts
5. Run the script:
python phishing_detecting.py
